# Металлист (MetCalc)

Нативное Android-приложение: калькулятор массы металлопроката + справочник допусков и посадок ISO 286. Без рекламы, оффлайн, тёмная тема.

## Возможности

- 11 форм сечений: круг, труба круглая, квадрат, шестигранник, лист/полоса, гнутый швеллер, труба прямоугольная, уголок равнополочный, двутавр, швеллер, арматура
- Два режима: «Расчёт» (свободные размеры) и «По ГОСТ» (типоразмер из справочника ГОСТ 8239 / 8240 / 8509 / 5781) — для двутавра, швеллера, уголка и арматуры
- Расчёт «по длине» и «по массе» (обратная задача)
- 23 предустановленных материала с плотностями, добавление своих
- Допуски и посадки ISO 286-1 (валы и отверстия, поля A–U/a–u/js/JS, квалитеты IT01–IT16)
- История расчётов с возможностью повтора и полного удаления

## Технологии

- Kotlin, Jetpack Compose, Material 3 (dynamic color на Android 12+)
- Clean Architecture: `domain` → `data` → `presentation`
- Hilt (DI), Room (БД), Navigation Compose
- minSdk 29 (Android 10), targetSdk 34
- kotlinx.serialization, DataStore

## Локальный запуск

Требуется JDK 17 и Android SDK.

```bash
./gradlew assembleDebug            # собрать debug APK
./gradlew testDebugUnitTest        # юнит-тесты
./gradlew installDebug             # установить на подключённое устройство
```

## Сборка через CI

GitHub Actions собирает debug APK автоматически на каждый push/PR в `main` — артефакт `metallist-debug-apk`. Релиз собирается на тег `v*.*.*` — артефакт `metallist-release` содержит APK и AAB.

## Источники справочных данных

- Плотности материалов: Марочник сталей и сплавов (В.Г. Сорокин), ГОСТ 5632/4784/613/15527/19281, справочники Точмех и e-metall
- ISO 286-1:2010 (стандартные допуски и основные отклонения)
- ГОСТ 8239-89 (двутавры), ГОСТ 8240-97 (швеллеры), ГОСТ 8509-93 (уголки), ГОСТ 5781-82 (арматура)
